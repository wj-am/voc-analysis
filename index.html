关键错误模式
⚡ 错误模式1：「卡」字关键词冲突
28条
「六折卡」「折扣卡」中的「卡」字被「客户端卡顿/掉线」规则单字关键词误触发，为最大单项错误。
资产流向查询 → 误判为 → 客户端卡顿/掉线 ×23
误操作诉求 → 误判为 → 客户端卡顿/掉线 ×3
常规活动规则 → 误判为 → 客户端卡顿/掉线 ×3
🔲 错误模式2：无意义 vs 其他问题 边界模糊
25条
短但有意义的输入（如「你好」「赠礼」「回声」）被 is_meaningless 规则误判。人工标准：仅纯「转人工」意图或纯情绪发泄才算无意义。
其他问题（有含义） → 误判为 → 无意义 ×25
🔀 错误模式3：渠道迁移 咨询 vs 申请
3条
「渠道迁移」「迁移」等宽泛词被「渠道服转官服申请(非小米)」命中。未明确表达申请/操作意图的应归为「迁移前信息咨询」。
迁移前信息咨询 → 误判为 → 渠道服转官服申请 ×3
💬 错误模式4：误操作/折扣卡口语化覆盖不足
~8条
「忘记使用六折卡」「金皮卡换错皮肤」「点错了」等口语化表达均被误判，当前规则缺少这些特征词。
误操作诉求 → 误判为 → 各类意图散落 ×8
🔑 错误模式5：账号注销 vs 换绑/解绑
3条
「注销」关键词优先级过高，覆盖了「换绑/解绑」意图。例：「注销掉他的话可以转过去吗」→ 应为换绑咨询。
账号换绑/解绑咨询 → 误判为 → 账号注销申请 ×3
👶 错误模式6：未成年退款政策 边界缺失
2条
未成年退款相关政策咨询被阶段1「处罚申诉」或「防沉迷其他」误判，需补充政策咨询类特征词。
未成年退款政策咨询 → 误判为 → 阶段1剔除 / 防沉迷其他 ×2
🛡️ 错误模式7：防沉迷关键词歧义
1条
同一关键词「防沉迷」同时命中「港澳台认证」和「未成年防沉迷规则」两个意图，需细化语义边界。
未成年防沉迷规则 → 误判为 → 防沉迷/港澳台认证 ×1
